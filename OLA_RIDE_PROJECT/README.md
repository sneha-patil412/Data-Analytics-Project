# 🚖 Ola Ride Data Analysis & Interactive Dashboard

An end-to-end data analytics project that processes ride-sharing data to extract business insights using SQL, Python (Streamlit), and Power BI.

## 📊 Live Project Overview
This project transforms raw ride data from July into a functional web application. It allows users to execute predefined SQL queries to answer specific business questions and view interactive visualizations.

### 🚀 Key Features
* **SQL Query Engine:** 10+ professional SQL insights executed via DuckDB.
* **EDA Visualizations:** Peak hour analysis, vehicle popularity, and booking status trends.
* **Power BI Integration:** An embedded interactive dashboard for deep-dive metrics.
* **Cleaned Data Pipeline:** Automated data cleaning (Date conversion, NULL handling, and Rating normalization).

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Database Logic:** SQL (DuckDB)
* **Web Framework:** Streamlit
* **Visualization:** Matplotlib, Power BI
* **Data Manipulation:** Pandas, Openpyxl
